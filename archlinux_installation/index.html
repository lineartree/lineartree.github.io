Non-UEFI System

Creating filesystem for non-UEFI system For non-UEFI system, you only have one single root partition. So just make it ext4:

1

mkfs.ext4 /dev/sda1

Mount to /mnt

1

mount /dev/sda3 /mnt

1

pacstrap /mnt base linux linux-firmware vim

1
2

genfstab -U /mnt >> /mnt/etc/fstab
arch-chroot /mnt

Install Grub Boot Loader

UEFI System

1
2
3
4
5

pacman -S grub efibootmgr
mkdir /boot/efi
mount /dev/sda1 /boot/efi
grub-install --target=x86_64-efi --bootloader-id=GRUB --efi-directory=/boot/efi
grub-mkconfig -o /boot/grub/grub.cfg

Non-UEFI System

Install grub on Non-UEFI systems Install grub package first:

1

pacman -S grub

And then install grub like this (donâ€™t put the disk number sda1, just the disk name sda):

1

grub-install /dev/sda

Last step:

1

grub-mkconfig -o /boot/grub/grub.cfg

1
2

## If you don't like to see any character at all not even while being entered set this to true.
